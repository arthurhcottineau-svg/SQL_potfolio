# 📊 SQL Portfolio - Analyse de Données
Bienvenue sur mon portfolio de Data Analyst ! Ce repository regroupe mes travaux sur la manipulation de bases de données relationnelles complexes.

## 🚌 Projet : Réseau de Bus d'Édimbourg (SQLZoo Challenge)
Ce projet démontre ma capacité à gérer des structures de données imbriquées et des jointures multiples.

### Compétences clés démontrées :
* **Self-Joins :** Utilisation de multiples instances de la même table pour modéliser des trajets avec correspondances.
* **Jointures multiples :** Connexion de tables de routes et de stations pour traduire des IDs en noms lisibles.
* **Logique de filtrage :** Utilisation de clauses `WHERE` complexes et de `DISTINCT` pour optimiser les résultats.

La requête dans `edinburgh_bus_transport.sql` permet de trouver tous les trajets impliquant deux bus entre 'Craiglockhart' et 'Lochend', en identifiant l'arrêt de transfert.
